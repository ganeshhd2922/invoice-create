# Invoice Generator 🧾

A simple React.js app to generate, preview, and download invoices as PDFs.

## 🚀 Features

- Add sender/client details
- Dynamic item list with price and quantity
- Auto total calculation
- Download invoice as PDF

## 🛠️ Tech Stack

- React.js
- HTML/CSS
- jsPDF
